# GAME ANALYSIS
---
Проект выполнен в рамках курса по Data Science (Яндекс Практикум)

### Цели и задачи проекта:
В проекте используется датасет `new_games.csv`, который содержит информацию о продажах игр, сделанных в разных жанрах и выпущенных на разных платформах, а также пользовательские и экспертные оценки игр.

__Задача__ — познакомиться с данными, проверить их корректность и провести предобработку, получив необходимый срез данных.
